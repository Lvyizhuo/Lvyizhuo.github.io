# 贡献指南

感谢你关注 Lvyizhuo.github.io！

## 关于本项目

这是 **吕一卓 (Yizhuo Lv)** 的个人学术主页，基于 [Academic Pages](https://academicpages.github.io/) 模板，采用 Jekyll 静态站点框架构建。

## 如何贡献

欢迎通过以下方式参与：

### 报告问题
- 访问 [Issues 页面](https://github.com/Lvyizhuo/Lvyizhuo.github.io/issues) 提交问题
- 请清晰描述问题现象、复现步骤和环境信息

### 提交 PR
1. Fork 本仓库
2. 从 `main` 分支创建你的特性分支
3. 提交改动（请写清晰的 commit message）
4. 向 `main` 发起 Pull Request

### 本地调试
详见 [README.md](README.md#-快速开始--本地开发) 中的本地开发指南。
